# About Project

## Aplikasi Forum Diskusi Sederhana
dimana aplikasi harus dapat melakukan:

- Membuat topik baru (CRUD).
- Membalas komentar.
- Memberikan vote pada postingan.
- User dengan role (Admin) dapat menghapus postingan atau memblokir pengguna.
